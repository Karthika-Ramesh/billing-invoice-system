# 🧾 Smart Billing & Invoice System

## 📌 Project Overview

The **Smart Billing & Invoice System** is a web-based application designed to generate invoices efficiently. It allows users to add items, calculate totals automatically, and generate professional invoices with GST and discount calculations.

---

## 🚀 Features

* 🔐 User Login (Frontend Authentication)
* ➕ Add / Delete Items
* 💰 Automatic Price Calculation
* 📊 GST (18%) Calculation
* 🎯 Discount (5%) Applied
* 🧾 Invoice Generation
* 🖨️ Print Invoice Option
* 📥 Download Invoice as PDF
* 💾 Data stored using LocalStorage
* 🎨 Modern Responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage (Browser Storage)
* html2pdf.js (for PDF download)
